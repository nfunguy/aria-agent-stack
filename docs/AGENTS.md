# ARIA Agent Specifications

**Author**: Niloy Sengupta, VP Consult, US Financial Services, Kyndryl
**Version**: 1.0

## Active Agents
1. **rental_audit**: Applies 75% vacancy factor to gross rent and subtracts PITI/A.
2. **dti_impact**: Recalculates borrower DTI based on corrected net rental income.
3. **classify_defect**: Assigns defect codes (PASS, RENT-001, RENT-002).
4. **generate_report**: Builds human-readable Chain of Thought explanations.
