# ARIA Deployment Guide

**Author**: Niloy Sengupta, VP Consult, US Financial Services, Kyndryl
**Version**: 1.0

## Local Development
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Run `docker compose up --build`
4. Access dashboard at `http://localhost:8501`

## Cloud Deployment
- Backend deployed to Render.com (Free Tier)
- Frontend deployed to Streamlit Community Cloud
